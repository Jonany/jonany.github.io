As I was thinking about this post earlier today, I thought for a split second that I had mislabeled these updates to be twice-a-week instead of twice-a-month but a quick DuckDuckGo search alleviated my fears -- sort of. According to Wikipedia, biweekly can mean either twice a week or twice a month. So I'm christening these updates, Fortnightly. :laughing: 

### What I'm learning

**Last week**, I spent some time learning about REST and playing with GitHub's [RESTful API][6]. Two resources I found particularly helpful were [these][1] [two][2] LinkedIn Learning courses. The first two chapters of *Designing RESTful APIs* has great information for APIs in general. I recommend watching it even if you're not working with RESTful APIs.

**This week**, I'm going to spend some time learning the basics of Python and using it to analyze data from my Trello boards using [Trello's RESTful API][3]. *[Crash Into Python][4]* is going to be my first step. I've been using a [Trello board][5] to keep track of my annual goals and daily habits and I want to start parsing that data to gain insights from it. Eventually, I want to create an interface that anyone can use to gain insights from their Trello data. *Side note: it looks like [Ollert][7] may already fulfill that need.*

**Today**, I [learned][8] more about GraphQL. I had heard a bit about it before but now I have a better understanding of what it is and what it is used for. It's essentially a query language for APIs. Its structure is very concise and declarative, two things that I'm a big fan of.

**On a less tech related note**, I started using [Blender][9] again and I'll be watching Blender Guru's *Blender 2.8* [series][10] this weekend to get up to speed with the new interface. It looks pretty slick! My goal is to create a 3D model of the basselope from Berkeley Breathed's [*The Last Basselope*][11]. It's a really cute story and the basselope has some interesting features that will be fun to model.

**On even less tech related note**, after about an hour of unsuccessfully trying to find Vietnamese movies and/or TV shows on YouTube that had subtitles in Vietnamese, I decided to call it quits for the day. Most of the videos I found were either had Vietnamese subs for non-Vietnamese words or had English subs. Reading subs in the language you're learning while hearing the words being spoken really helps you make connections between the two. My next step is to see if I can find any Vietnamese lyrics videos. That won't be quite as good as conversations but it's better than nothing.
